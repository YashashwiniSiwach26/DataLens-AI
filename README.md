# DataLens AI

## 📌 Overview
DataLens AI is an intelligent dataset analysis platform that helps users understand the quality of their datasets before training machine learning models.

## 🎯 Goal
- Analyze datasets
- Detect quality issues
- Recommend preprocessing steps
- Predict ML readiness
- Explain model behavior

## 🛠️ Tech Stack
- Python
- NumPy
- Pandas
- Scikit-learn
- FastAPI
- Docker
